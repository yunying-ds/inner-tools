# Inner Tools - Sedona Method Release App

🌐 **Live**: https://www.yunying-inner-tools.site

A web app that guides users through the Sedona Method - a 9-step emotional release process.

---

## What It Does

**9-Step Release Flow:**
1. Emotion identification (AI-powered, multi-emotion support)
2. Allow the feeling to exist
3-5. Three questions: Could you let it go? Would you? When?
6. Intensity rating (1-10 scale, auto-loops if high)
7. Identify core wants (approval/control/security)
8. Release the want
9. Reflect and continue or complete

**Key Features:**
- Multi-emotion queue processing
- Intelligent looping (max 3 attempts, stuck detection)
- "Reframe wants" for alternative perspectives
- Optional journaling with pre-fill
- Session history

## Key Decisions

| Problem | Solution |
|---------|----------|
| Hydration errors | `ssr: false` + client component split |
| Multi-emotion handling | Queue-based sequential processing |
| API parsing failures | Switched to `tool_use` pattern |
| Infinite loops | MAX_LOOPS=3 + "not improving" detection |
